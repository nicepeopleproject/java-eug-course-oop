Цель: предоставить унифицированный интерфейс вместо нескольких интерфейсов подсистемы.

Для чего используется:
Используется для определния интерфейса высоког оуровня, который упрощает использование подсистемы.

Примеры:
- изолирование клиентов от компонентов подсистемы, упрощаю работу с ней
- необходимость ослабления связанности подсистемы с клиентами
