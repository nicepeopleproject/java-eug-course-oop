Цель Динамическое добавление новые обязаностей объекту.


Используется в качестве альтернативы порождению подклассов для расширения функциональности

Примеры:
- динамическое и понятное клиентам добавление обязанностей объекту
- реализация обязанностей, которые могут быть сняты с объеты
- расширение класс патем порождения подклассов невозможно по каким -либо причинам
